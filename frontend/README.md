# Frontend

This project is responsible for showing the frontend web application through which the users will interact.

## Structure

```text
src/
├── api/          # HTTP clients
├── components/   # Reusable UI components
├── views/        # Pages
├── models/       # TypeScript domain models
├── router/       # Routing
├── stores/       # Pinia stores
└── utils/        # Utility functions
```

# State management

We use Pinia to manage the application state store.

# Dev get started

1. install dependencies
   ```bash
   cd frontend
   npm install
   ```
